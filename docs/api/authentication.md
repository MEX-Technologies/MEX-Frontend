# Authentication

How to authenticate with the MEX API.

## Current Status

**Note**: API authentication is in development. This guide will be updated when available.

## Planned Authentication Methods

### API Keys

Simple authentication using API keys:

```
Authorization: Bearer YOUR_API_KEY
```

### OAuth 2.0

Full OAuth 2.0 implementation for secure authentication.

### JWT Tokens

JSON Web Tokens for stateless authentication.

## Security Best Practices

- Keep API keys secret
- Use environment variables
- Rotate keys regularly
- Use HTTPS only
- Implement rate limiting
