# API Overview

MEX API documentation and integration guide.

## Introduction

The MEX API allows developers to integrate cross-border payment functionality into their applications.

## Current Status

**Note**: The MEX API is currently in development and not yet publicly available. This documentation will be updated as we approach launch.

#### Authentication

#### Transactions

#### API Key Authentication

Elegant handling of rate limit exceeded scenarios
