# Markdown Guide

How to use markdown in MEX documentation and blog posts.

## Overview

MEX uses markdown for all documentation and blog content. This guide covers the markdown features available.

## Basic Markdown

### Headers

```markdown
# Heading 1
## Heading 2
### Heading 3
```

### Text Formatting

```markdown
**Bold text**
*Italic text*
`inline code`
```

### Lists

```markdown
- Bullet point
- Another point

1. Numbered item
2. Another item
```

### Links

```markdown
[Link text](https://example.com)
```

### Code Blocks

\`\`\`
Code block
\`\`\`
